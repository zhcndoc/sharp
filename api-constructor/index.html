Section titled “new”

new Sharp([input], [options])

构造函数工厂，用于创建 sharp 的实例，随后可以链式调用其他方法。

可以从此对象流出 JPEG、PNG、WebP、GIF、AVIF 或 TIFF 格式的图像数据。当使用基于流的输出时，可以从 info 事件中获得派生属性。

在处理过程中遇到的非关键问题将作为 warning 事件发出。

实现了 stream.Duplex 类。

当加载多个页面/帧的动画图像时，这些将组合成一个垂直堆叠的“厕纸卷”图像，其整体高度为 pageHeight 乘以 pages 的数量。

抛出:

  • Error 无效参数
参数类型默认描述
[input]Buffer | ArrayBuffer | Uint8Array | Uint8ClampedArray | Int8Array | Uint16Array | Int16Array | Uint32Array | Int32Array | Float32Array | Float64Array | string | Array如果提供，可以是包含 JPEG、PNG、WebP、AVIF、GIF、SVG 或 TIFF 图像数据的 Buffer / ArrayBuffer / Uint8Array / Uint8ClampedArray，或包含原始像素图像数据的 TypedArray，或包含指向 JPEG、PNG、WebP、AVIF、GIF、SVG 或 TIFF 图像文件的文件系统路径的字符串。也可以提供输入数组，这些输入会被拼接在一起。未提供时，可以将 JPEG、PNG、WebP、AVIF、GIF、SVG、TIFF 或原始像素图像数据通过流输入到对象中。
[options]Object如果提供，则为带有可选属性的对象。
[options.failOn]string“‘warning‘“何时中止对无效像素数据的处理，可选值按敏感度从低到高依次为：‘none’、‘truncated’、‘error’、‘warning’。更高等级包含更低等级。无效元数据总会中止。对不受信任的输入请使用默认的 ‘warning’ 等级。
[options.limitInputPixels]number | boolean268402689不处理像素总数（宽 x 高）超过此限制的输入图像。假定输入元数据中包含的图像尺寸是可信的。像素总数的整数值，设为 zero 或 false 可移除限制，设为 true 可使用默认限制 268402689（0x3FFF x 0x3FFF）。
[options.limitInputChannels]number | boolean5不处理通道数超过此限制的输入图像。假定图像元数据是可信的。通道数的整数值，设为 zero 或 false 可移除限制，设为 true 可使用默认限制 5。
[options.unlimited]booleanfalse将其设为 true 可移除有助于防止内存耗尽的安全特性（JPEG、PNG、SVG、HEIF）。
[options.autoOrient]booleanfalse将其设为 true 可旋转/翻转图像以匹配 EXIF Orientation（如果存在）。
[options.sequentialRead]booleantrue将其设为 false 可使用随机访问而不是顺序读取。某些操作会自动这样做。
[options.density]number72渲染 SVG 和 PDF 图像时使用的 DPI，范围为 1 到 100000。
[options.ignoreIcc]numberfalse是否忽略嵌入的 ICC 配置文件（如果有）。
[options.pages]number1多页输入（GIF、WebP、TIFF）要提取的页数，全部页数使用 -1。
[options.page]number0多页输入（GIF、WebP、TIFF）从第几页开始提取，基于零。
[options.animated]booleanfalse将其设为 true 可读取动画图像（GIF、WebP、TIFF）的所有帧/页，相当于将 pages 设为 -1。
[options.raw]Object描述原始像素输入图像数据。像素排列请参见 raw()。
[options.raw.width]number像素宽度的整数值。
[options.raw.height]number像素高度的整数值。
[options.raw.channels]number通道数的整数值，介于 1 和 4 之间。
[options.raw.premultiplied]boolean指定原始输入已经预乘，设为 true 可避免 sharp 对图像进行预乘。（可选，默认 false）
[options.raw.pageHeight]number动画图像每一页/帧的像素高度，必须是 raw.height 的整数因子。
[options.create]Object描述要创建的新图像。
[options.create.width]number像素宽度的整数值。
[options.create.height]number像素高度的整数值。
[options.create.channels]number通道数的整数值，取 3（RGB）或 4（RGBA）。
[options.create.background]string | Object由 color 模块解析，以提取红、绿、蓝和 alpha 的值。
[options.create.pageHeight]number动画图像每一页/帧的像素高度，必须是 create.height 的整数因子。
[options.create.noise]Object描述要创建的噪声。
[options.create.noise.type]string生成噪声的类型，目前仅支持 gaussian。
[options.create.noise.mean]number128生成噪声中像素值的均值。
[options.create.noise.sigma]number30生成噪声中像素值的标准差。
[options.text]Object描述要创建的新文本图像。
[options.text.text]string要渲染的 UTF-8 字符串文本。可以包含 Pango 标记，例如 <i>Le</i>Monde。
[options.text.font]string要使用的字体名称。
[options.text.fontfile]string可供 font 使用的字体文件的绝对文件系统路径。
[options.text.width]number0用于自动换行的像素整数宽度。宽于此值的文本行将按单词边界断行。
[options.text.height]number0最大像素高度整数。定义后，dpi 将被忽略，文本会自动适配由 width 和 height 定义的像素分辨率。若未指定 width 或将其设为 0，则会被忽略。
[options.text.align]string“‘left‘“多行文本的对齐样式（'left'、'centre'、'center'、'right'）。
[options.text.justify]booleanfalse将其设为 true 可对文本应用两端对齐。
[options.text.dpi]number72渲染文本时的分辨率（尺寸）。若指定了 height，则不生效。
[options.text.rgba]booleanfalse将其设为 true 可启用 RGBA 输出。这对于彩色表情符号渲染，或支持诸如 <span foreground="red">Red!</span> 之类的 pango 标记功能很有用。
[options.text.spacing]number0文本行高，单位为 point。若未指定，则使用字体行高。
[options.text.wrap]string“‘word‘“提供宽度时的自动换行样式，可选：‘word’、‘char’、‘word-char’（优先按单词，失败时按字符）或 ‘none’。
[options.join]Object描述数组中的输入图像应如何连接。
[options.join.across]number1水平连接的图像数量。
[options.join.animated]booleanfalse将其设为 true 可将这些图像连接为动画图像。
[options.join.shim]number0在连接图像之间插入的像素数。
[options.join.background]string | Object由 color 模块解析，以提取红、绿、蓝和 alpha 的值。
[options.join.halign]string“‘left‘“水平连接图像的水平对齐样式（'left'、'centre'、'center'、'right'）。
[options.join.valign]string“‘top‘“垂直连接图像的垂直对齐样式（'top'、'centre'、'center'、'bottom'）。
[options.tiff]ObjectTIFF 特定选项的描述。
[options.tiff.subifd]number-1为 OME-TIFF 提取的子图像文件目录，默认使用主图像。
[options.svg]ObjectSVG 特定选项的描述。
[options.svg.stylesheet]stringSVG 输入的自定义 CSS，在 CSS 级联中使用用户来源应用。
[options.svg.highBitdepth]booleanfalse将其设为 true 可将 SVG 输入渲染为每通道 32 位（总计 128 位）的 RGBA，而不是每通道 8 位（总计 32 位）。
[options.pdf]ObjectPDF 特定选项的描述。需要使用全局安装的、编译时支持 PDFium、Poppler、ImageMagick 或 GraphicsMagick 的 libvips。
[options.pdf.background]string | Object当 PDF 部分透明时使用的背景颜色。由 color 模块解析，以提取红、绿、蓝和 alpha 的值。
[options.openSlide]ObjectOpenSlide 特定选项的描述。需要使用全局安装的、编译时支持 OpenSlide 的 libvips。
[options.openSlide.level]number0要从多层输入中提取的层级，基于零。
[options.jp2]ObjectJPEG 2000 特定选项的描述。需要使用全局安装的、编译时支持 OpenJPEG 的 libvips。
[options.jp2.oneshot]booleanfalse将其设为 true 可一次性解码平铺的 JPEG 2000 图像，以提高兼容性。

Section titled “clone”

clone() ⇒ Sharp

快照 Sharp 实例，返回一个新的实例。克隆的实例继承其父实例的输入。这使得多个输出流，并因此多个处理管道能够共享单个输入流。
